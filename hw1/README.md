# RepetitorGo — Uber для репетиторов

**Бизнес-модель:** C2C

---

## Elevator Pitch

Ученики и родители, которым нужен репетитор, тратят дни на поиск проверенного специалиста через сарафанное радио и объявления. Мы — платформа, где частные преподаватели предлагают уроки напрямую, а ученики находят и бронируют занятие за несколько минут. В отличие от обычных досок объявлений, каждый репетитор проходит верификацию, а оплата и расписание — прямо в приложении. В результате репетиторы получают стабильный поток учеников, а ученики — проверенного специалиста без лишних звонков и рисков.

## Lean Canvas

| Блок | Описание |
|------|----------|
| **1. Сегменты потребителей** | Школьники и студенты, готовящиеся к экзаменам (ЕГЭ, ОГЭ, сессии); родители, ищущие репетиторов для детей; частные преподаватели и студенты старших курсов |
| **2. Проблема** | Сложно найти проверенного репетитора быстро; нет прозрачности по ценам и отзывам; неудобное согласование расписания и оплаты вручную; репетиторы тратят время на поиск учеников |
| **3. Уникальная ценность** | Платформа для мгновенного поиска верифицированного репетитора с онлайн-записью и оплатой — как вызов такси, только для знаний |
| **4. Решение** | Приложение с каталогом репетиторов, фильтрами (предмет, цена, рейтинг, онлайн/офлайн), встроенным календарём и оплатой; верификация преподавателей по диплому и пробному уроку |
| **5. Каналы** | ВКонтакте и Telegram-реклама, партнёрство со школами и университетами, SEO/контекст (Яндекс.Директ), реферальная программа среди учеников |
| **6. Потоки прибыли** | Комиссия с каждого урока (10–15%); платное продвижение анкеты репетитора; подписка «Премиум» для репетиторов (приоритет в выдаче, аналитика) |
| **7. Структура издержек** | Разработка и поддержка приложения, реклама, верификация репетиторов, служба поддержки, платёжная инфраструктура |
| **8. Ключевые метрики** | Количество успешно проведённых уроков, процент повторных записей, средний рейтинг репетиторов, время от регистрации до первого урока |
| **9. Скрытое преимущество** | Система верифицированных отзывов и история уроков, которую невозможно подделать; сетевой эффект — чем больше репетиторов, тем быстрее находится нужный специалист |

## Как это работает

1. **Репетитор** (C2C) регистрируется, проходит верификацию (диплом + пробный урок), указывает предметы, цену и расписание
2. **Ученик** находит подходящего репетитора через фильтры, смотрит отзывы и бронирует урок онлайн
3. **Платформа** обеспечивает безопасную оплату, напоминания и разрешение споров

## Существующие альтернативы

- Profi.ru / YouDo — нет верификации по знаниям, слабая система расписания
- Авито / ВКонтакте объявления — нет гарантий, оплата наличными, неудобно
- Школьные репетиторы по знакомству — ограниченный выбор, нет прозрачности цен
